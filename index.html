Superfastduplicatefinderfinder

Very simple and superfast duplicate file finder.

Download this project as a .zip file Download this project as a tar.gz file

This is a very simple duplicate file finder which uses only two parameters file name and file size to find duplicates, which makes it lightning fast. It is particularly useful when you have a huge amount of disk files and you want a super-fast tool to find the duplicates. Very soon I am going to add features for optional better accuracy with checksum calculation.
